# Tools

## Git

We use [Git](https://git-scm.com/) for our souce control. Follow the guide on
[GitHub's Git Guides](https://github.com/git-guides/install-git) for thorough instructions on how to install Git on
Windows, MacOS and Linux.

## Docker / Podman

We use Podman as our container engine - visit their website at https://podman-desktop.io/ and follow their instructions
to install Podman desktop.

:::note
To run `podman compose`, you must perform additional set up, refer to
the following link for details: https://podman-desktop.io/docs/compose/running-compose
:::
