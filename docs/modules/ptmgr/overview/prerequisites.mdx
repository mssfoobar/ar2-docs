# Prerequisites and Dependencies

## Assumptions

This document assumes that
- You are familiar with IAMS module and have working IAMS installed in your local machine.
- You have read access to AGIL Ops Hub container registry.
- You are familiar with Firebase Cloud Messaging(FCM) for mobile push notification.

## Prerequisites
- Have working `Podman Desktop` or `Docker Engine` on your local machine.
- Have read access to AGIL Ops Hub container registry.
- Have IAMS module installed in your local machine.
- Have Firebase Cloud Messaging(FCM) client set up in your mobile application.

:::note
To learn how to set up FCM, refer to
[Google FCM guide.](https://firebase.google.com/docs/cloud-messaging/android/client). You can also refer to their
quickstart guide [here](https://github.com/firebase/quickstart-android/tree/master/messaging).
:::

## Dependencies

### AOH modules
- IAMS

### Others
- Postgres
- Firebase Cloud Messaging(FCM)
